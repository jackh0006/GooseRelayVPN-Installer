# Contributing

Thank you for helping improve GooseRelayVPN Installer.

## Development

```bash
git clone https://github.com/jackh0006/GooseRelayVPN-Installer.git
cd GooseRelayVPN-Installer
```

## Pull Requests

1. Fork repository
2. Create feature branch
3. Commit changes
4. Submit pull request

## Guidelines

- ShellCheck clean
- No hardcoded secrets
- Proper error handling
- Keep documentation updated
